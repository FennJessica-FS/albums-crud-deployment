
# Albums CRUD Deployment (MERN)

## Tech Stack
- React (Frontend)
- Express (Backend)
- MongoDB (Database)
- Render (API Hosting)
- Netlify (Frontend Hosting)

## Features
- Full CRUD functionality (GET, POST, PUT, DELETE)
- Two-page React interface (Home + Albums + About)
- Custom CSS styling modes
- RESTful API with MongoDB model

## Model Fields
- title (String)
- artist (String)
- year (Number)
- createdAt (Timestamp)
